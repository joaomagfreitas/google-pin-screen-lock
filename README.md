# google-pin-screen-lock

Google's pin screen lock implemented using different frameworks. Feel free to reuse any of the code.

# Frameworks

- [ ] Android XML (API 17)
- [ ] Android Jetpack Compose (API 21)
- [ ] iOS SwiftUI (iOS 15.5)
- [ ] Flutter
- [ ] Web (Pure HTML/CSS)
